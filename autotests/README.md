# Registration Automation Project

This project automates the registration form on the DemoQA website using Python, pytest, Playwright, and Allure.

## Project Structure
```
autotests/
├── README.md
├── pages
│   ├── base_page.py
│   └── registration_page.py
├── pytest.ini
├── requirements.txt
├── test_data
│   └── test_data.json
├── tests
│   └── test_registration.py
└── utilities
    ├── conftest.py
    ├── helpers.py
    └── logger.py
```
